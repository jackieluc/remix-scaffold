# Remix Scaffold

- [Remix Docs](https://remix.run/docs)

## Project Setup

Largely inspired by Remix's Indie Stack, but simpler.

- create-remix@latest
- Tailwindcss
- Vitest
- Playwright
- Eslint
- Prettier
- Vercel deployment target

## Development

Install
```sh
npm install
```

Start dev server
```sh
npm run dev
```

Test with Vitest
```sh
npm run test
```

Test coverage
```sh
npm run coverage
```

E2E test with Playwright
```sh
npm run e2e
```

Build
```sh
npm run build
```

Lint
```sh
npm run lint
```

Prettier
```sh
npm run format
```
